# Warlock Simulator TBC
 
 Warlock simulator for WoW TBCC written in C++ & Javascript
 
 ## Building Locally
 ### Requirements:
 [nvm-windows](https://github.com/coreybutler/nvm-windows/releases)  
 `nvm install 14.15.5`  
 `nvm use 14.15.5`  
 `npm install`
 ### Run
 Run Frontend: `gulp`  
 Compile C++: `make`
